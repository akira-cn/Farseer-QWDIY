Dom  >  Dom.create 按html创建dom节点。

语法：

Dom.create(html, rfrag, doc)

参数：

类型描述
htmlstringcssSelector字符串
rfragboolean(Optional) 是否返回碎片。为true时返回碎片，否则返回第一个创建的第一个节点
docDocument(Optional) Document对象。不传时，默认用当前document创建元素

返回值：

Element|DocumentFragment
