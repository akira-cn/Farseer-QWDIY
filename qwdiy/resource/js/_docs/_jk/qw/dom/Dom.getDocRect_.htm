Dom  >  Dom.getDocRect 获取doc的一些坐标信息。

语法：

Dom.getDocRect(doc)

参数：

类型描述
docDocument(Optional) Document对象。不传时，默认是当前document

返回值：

Object 其格式为： 

{
	width : w,
	height : h,
	scrollWidth : scrollW,
	scrollHeight : scrollH,
	scrollX : scrollX,
	scrollY : scrollY
}
