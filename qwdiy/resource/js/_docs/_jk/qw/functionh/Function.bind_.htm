Function  >  Function.bind 固化一个func里的this，以及某些参数，返回新函数。

语法：

Function.bind(func,thisObj,arg0)

参数：

类型描述
funcFunction函数对象
thisObjany将func里的this固化成thisObj
args0any(Optional) 固化func的第一个参数
args1any(Optional) 固化func的第二个参数
argsnany(Optional) 固化func的第n+1个参数

返回值：

function 返回新的function
