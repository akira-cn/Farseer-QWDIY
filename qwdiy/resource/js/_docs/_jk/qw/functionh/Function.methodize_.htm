Function  >  Function.methodize 将一个满足Helper规范的静态方法，变换成一个原型方法，即，静态方法的第一个参数，变成了this。

语法：

Function.methodize(func,attr)

参数：

类型描述
funcfunction静态函数
attrstring(Optional) 如果有此值，静态方法的第一个参数，将变成this[attr]

返回值：

function 返回新的function
