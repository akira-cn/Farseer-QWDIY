W.query/W.one方法支持各种组合语法。
如：
  input,textarea,select
  #id>div
  #id~div.bbb

w.ancestorNode/W.nextSibling等只支持最简selector
即没有“, >~+”关系符的selector，如
  div.aaa
  input[readOnly]

w.delegate支持以“>”或“ ”打头的selector
如：
  >*
  >*>*
  ul.aaa a.op_del

w.delegate不支持以“+”或“~”打头的selector
