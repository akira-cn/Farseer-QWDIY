NodeW  >  w.insert 插入元素

语法：

w.insert(sWhere, newEl)

参数：

类型描述
sWherestring 指定插入元素的地方
  1. beforebegin : 插入到元素前面
  2. afterbegin : 插入到元素内部的开头
  3. beforeend : 插入到元素内部的末尾
  4. afterend : 插入到元素后面
newElHTMLElement要插入的元素
注：此处不可插入DocumentFragment

返回值：

w本身
