Object  >  ObjectH.get 静态方法 获得对象的某个属性值

语法1：

ObjectH.get(obj,prop,nullSensitive)

参数：

类型描述
objobject源对象
propstring属性名，深层属性用“.”区分表示，例如：style.color
nullSensitiveboolean(Optional) 是否对属性链异常敏感。即，如果属性链中间为空，是否抛出异常

返回值：

返回属性值。

语法2：

ObjectH.get(obj,func,args1)

参数：

类型描述
objobject源对象
funcfunctiongetter函数
args1any(Optional) getter所需要的第二个参数。
argsnany(Optional) getter所需要的第(n+1)个参数。

返回值：

返回getter函数运行的返回值。

语法3：

ObjectH.get(obj,props,nullSensitive)

参数：

类型描述
objobject源对象
propsArray属性名数组
nullSensitiveboolean(Optional) 是否对属性链异常敏感。即，如果属性链中间为空，是否抛出异常

返回值：

Array 反回与"属性名数组"对应的"属性值数组"。
