g 静态方法 获取原生Element

语法1：

g(id,doc)

参数：

类型描述
idstring元素id
docDocument(Optional) Document对象。不传时，默认在当前document下按id查询

返回值：

浏览器原生Element。如果些id找不到，则反回null

语法2：

g(html,doc)

参数：

类型描述
htmlstring以“<”打头的字符串
docDocument(Optional) Document对象。不传时，默认用当前document创建元素

返回值：

根据html字符串，生成html元素，并返回该元素。

语法3：

g(oWrap)

参数：

类型描述
oWrapWrapWrap对象

返回值：

返回g(oWrap.core)

语法4：

g(oAny)

参数：

类型描述
oAny非string非Wrapstring与Wrap以外的其它类型

返回值：

返回oAny
